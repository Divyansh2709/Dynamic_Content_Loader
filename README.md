# 🚀 Dynamic Content Loading Web App

> A modern full-stack web application demonstrating dynamic content loading using PHP, MySQL, and JavaScript with a premium UI/UX.

## ✨ Features

### 🔍 Smart Search
- Live search with debouncing
- Efficient API usage
- Instant UI updates

### 📂 Advanced Filtering
- Filter posts by category
- Dynamic category loading from database

### ⏱ Sorting System
- Sort posts by:
  - Newest
  - Oldest

### 📄 Pagination
- Server-side pagination
- Smooth navigation between pages

### 🌙 Dark Mode
- Toggle between light and dark themes
- Persistent theme using localStorage

### ⚡ Dynamic Loading
- No page reloads
- Powered by Fetch API
- Seamless UX

### 🧠 URL State Management
- Shareable URLs
- Maintains filters, search, and pagination

### 💎 Premium UI/UX
- Glassmorphism design
- Gradient accents
- Skeleton loaders
- Smooth animations
